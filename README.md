# Development

## Setup

### Venv
Start pyenv:
```sh
eval "$(pyenv init -)"
eval "$(pyenv virtualenv-init -)"
```

Make venv:
```sh
pyenv virtualenv 3.10.9 (cat .pyenv) && \
  pyenv activate (cat .pyenv)
```

### Pip
```sh
python3.10 -m pip install --upgrade pip
```

### Install Poetry
```sh
pip install poetry
```
